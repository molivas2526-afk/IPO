# Projecte Free: Solució Integral per a Gestió Veterinària i Videojoc 2D

## Descripció del Projecte
El **Projecte Free** és una solució tecnològica transversal que combina el desenvolupament de programari, la gestió empresarial i la configuració de sistemes. L'objectiu és oferir una eina completa per a una clínica veterinària, integrant des d'un videojoc interactiu fins a la gestió administrativa mitjançant un ERP/CRM.

## Tecnologies Utilitzades
* **Backend & Joc:** Java (Llibreria Swing per a la interfície 2D).
* **Bases de Dades:** SQL amb l'ús de procediments i triggers per a l'automatització.
* **Web:** HTML5, CSS3 (Flexbox/Grid) i JavaScript per a una Landing Page adaptativa.
* **Gestió Empresarial:** Dolibarr (ERP/CRM) per a control d'estoc, vendes i agenda.
* **Sistemes:** Oracle VirtualBox per al desplegament de servidors Linux i Windows.
* **Control de Versions:** GitHub.

## Funcionalitats Principals
* **Videojoc 2D:** Interfície funcional amb accés a base de dades.
* **Gestió de Clínica:** Digitalització integral de clients, visites i vendes amb Dolibarr.
* **Automatització:** Implementació de lògica de negoci directament a la base de dades.
* **Landing Page:** Presentació del producte amb disseny modern i "mobile-first".

# Projecte Free: Solució Integral per a Gestió Veterinària i Videojoc 2D

## Descripció del Projecte
El **Projecte Free** és una solució tecnològica transversal que combina el desenvolupament de programari, la gestió empresarial i la configuració de sistemes. [cite_start]L'objectiu és oferir una eina completa per a una clínica veterinària, integrant des d'un videojoc interactiu fins a la gestió administrativa mitjançant un ERP/CRM[cite: 96, 98].

## Tecnologies Utilitzades
* **Backend & Joc:** Java (Llibreria Swing per a la interfície 2D)[cite: 96, 100].
* **Bases de Dades:** SQL amb l'ús de procediments i triggers per a l'automatització[cite: 100].
* **Web:** HTML5, CSS3 (Flexbox/Grid) i JavaScript per a una Landing Page adaptativa[cite: 19, 100].
* **Gestió Empresarial:** Dolibarr (ERP/CRM) per a control d'estoc, vendes i agenda[cite: 71, 101].
* **Sistemes:** Oracle VirtualBox per al desplegament de servidors Linux i Windows[cite: 97, 101].
* **Control de Versions:** GitHub[cite: 101].

## Funcionalitats Principals
* **Videojoc 2D:** Interfície funcional amb accés a base de dades[cite: 96].
* **Gestió de Clínica:** Digitalització integral de clients, visites i vendes amb Dolibarr.
* **Automatització:** Implementació de lògica de negoci directament a la base de dades[cite: 96, 104].
* **Landing Page:** Presentació del producte amb disseny modern i "mobile-first"[cite: 97].

## El meu Rol i Contribució
Com que es tracta d'un projecte individual, he assumit el cicle de vida complet del programari[cite: 104]:
1.  **Disseny:** Creació de diagrames de casos d'ús i arquitectura de dades[cite: 96, 104].
2.  **Desenvolupament:** Codificació del joc en Java i la interfície web[cite: 104, 105].
3.  **Sistemes:** Configuració i manteniment dels servidors que allotgen la solució[cite: 104].
4.  **Documentació:** Redacció tècnica i presentació professional[cite: 105].
